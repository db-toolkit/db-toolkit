# Welcome to DB Toolkit

DB Toolkit is an [open-source](https://github.com/Adelodunpeter25/db-toolkit) cross-platform desktop database management application built with Electron, React, and Python FastAPI. It supports PostgreSQL, MySQL, SQLite, and MongoDB.

Manage your databases with a modern, intuitive interface that combines powerful query editing, visual schema exploration, and automated backup management. Whether you're a developer, DBA, or data analyst, DB Toolkit streamlines your database workflow.

Check out [Getting Started](./gettingstarted) for installation instructions.
