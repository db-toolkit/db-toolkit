# Getting Started

## Installation

Download the latest version for your platform:

- **macOS** - Download .dmg, open and drag to Applications
- **Windows** - Download .exe and run the installer
- **Linux** - Download .AppImage (see steps below)

### Linux Installation

AppImage requires FUSE to run. Follow these steps:

**Ubuntu/Debian:**
```bash
sudo apt install libfuse2
chmod +x DB-Toolkit-linux.AppImage
./DB-Toolkit-linux.AppImage
```

**Arch Linux:**
```bash
yay -S db-toolkit-bin
https://github.com/BulusHamnu/db-toolkit-aur.git

```

**Fedora:**
```bash
sudo dnf install fuse
chmod +x DB-Toolkit-linux.AppImage
./DB-Toolkit-linux.AppImage
```

Alternatively, download the `.deb` package for Debian/Ubuntu:
```bash
sudo dpkg -i DB-Toolkit-linux.deb
```

### System Requirements

- macOS 10.13+ (Intel & Apple Silicon)
- Windows 10+
- Linux Ubuntu 18.04+, Fedora 32+

## First Launch

1. Open DB Toolkit
2. Click on the connections button in the sidebar
3. Click "Add Connection" to create your first database connection

## Create a Connection

Click **"Add Connection"** and fill in:

- **Name** - Friendly name (e.g., "Local PostgreSQL")
- **Type** - PostgreSQL, MySQL, MariaDB, SQLite, or MongoDB
- **Host** - Database server address
- **Port** - Database port
- **Database** - Database name
- **Username** - Your username
- **Password** - Your password

Click **"Test Connection"** to verify, then **"Save"**.
