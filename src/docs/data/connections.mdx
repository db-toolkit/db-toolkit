# Connection Management

## Adding a Connection

1. Click **"Add Connection"**
2. Fill in connection details
3. Optionally assign to a group
4. Click **"Test Connection"** to verify
5. Click **"Save"** to store the profile

## Connection Groups

Organize your connections into custom groups for better management:

1. Click **"Manage Groups"** button
2. Create new groups or edit existing ones
3. Assign connections to groups when creating/editing
4. View grouped connections on the main page

## Connection Types

### PostgreSQL
- Host: localhost
- Port: 5432

### MySQL
- Host: localhost
- Port: 3306

### SQLite
- Database Path: /path/to/database.db

### MongoDB
- Host: localhost
- Port: 27017

## Managing Connections

- **Edit** - Click edit icon to update details
- **Delete** - Click delete icon to remove
- **Test** - Verify credentials before saving
- **Groups** - Organize connections by environment or project
