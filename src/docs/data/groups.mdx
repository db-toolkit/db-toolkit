# Connection Groups

Connection Groups help you organize your database connections by environment, project, or any custom categorization that fits your workflow.

## Creating Groups

1. Navigate to the **Connections** page
2. Click the **"Manage Groups"** button
3. Enter a group name in the input field
4. Click the **"+"** button to create the group

## Assigning Connections to Groups

### When Creating a Connection
1. Fill in the connection details
2. Select a group from the **"Group (Optional)"** dropdown
3. Save the connection

### When Editing a Connection
1. Click the edit icon on any connection
2. Select or change the group assignment
3. Save the changes

## Managing Groups

### Edit Group Names
1. Open the **"Manage Groups"** sidebar
2. Click the edit icon next to any group
3. Update the name and click the checkmark

### Delete Groups
1. Open the **"Manage Groups"** sidebar
2. Click the delete icon next to any group
3. Confirm the deletion
4. Connections in deleted groups become ungrouped

## Group Display

### Main Connections Page
- **Group Cards** - Shows at the top with connection counts
- **Ungrouped Connections** - Listed below group cards
- **Search** - Includes both grouped and ungrouped connections

### Group Pages
- **Double-click** any group card to view only that group's connections
- **Search** - Filter connections within the specific group
- **Back Button** - Return to the main connections page

## Benefits

- **Organization** - Keep development, staging, and production connections separate
- **Quick Access** - Find connections faster with logical grouping
- **Project Management** - Group connections by client or project
- **Environment Separation** - Avoid accidentally connecting to wrong environments
