# Settings

## Appearance

### Dark Mode

- Automatic OS theme detection
- Manual toggle with Ctrl/Cmd+K
- Persists across sessions

### Theme Options

- Light mode
- Dark mode
- System (auto-detect)

## Query Defaults

### Row Limit

- Default: 1000 rows
- Range: 10 - 10,000
- Prevents loading millions of rows

### Timeout

- Default: 30 seconds
- Range: 5 - 300 seconds
- Cancels long-running queries

## Editor Preferences

### Font Size

- Range: 12px - 24px
- Default: 14px

### Tab Size

- 2 or 4 spaces
- Default: 2 spaces

### Word Wrap

- Enable/disable
- Default: Enabled

### Minimap

- Show/hide code overview
- Default: Shown

## Connection Defaults

### Default Ports

- PostgreSQL: 5432
- MySQL: 3306
- MongoDB: 27017

### Connection Timeout

- Default: 10 seconds
- Range: 5 - 60 seconds
